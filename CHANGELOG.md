# Changelog

All notable changes to `langgraph-cpp` will be documented in this file.

This project uses pre-1.0 preview releases. Source API, ABI, and persisted
schemas may change before a stable release; schema or public contract changes
should be called out explicitly.

## Unreleased

- Prepared open-source release hygiene: license, contribution guide, security
  policy, third-party notices, and CI workflow.
- Current project state is suitable for an alpha/developer-preview release, not
  a production-stable declaration.

## 0.1.0-alpha

- Initial planned public preview for the C++23 LangGraph-style edge runtime.
- Includes graph execution, state/reducers, checkpoint/resume, streaming,
  interrupt, mock model/tool loop, SQLite-backed storage, and edge adapter
  examples.
